
## 总体评价

**仓库组织和问题拆解合格，但当前第二问不能标记为 `done`。**

它解决了第一问“固定终止质量导致油耗恒定”的问题，改用固定航程比较，这一点正确。 但模型中存在两个实质性错误：

1. 所谓“完整非标准大气修正”实际上没有修正压力；
2. 当前采用等真空速策略，声速并未真正进入油耗模型，但结果中声称声速改变了燃油惩罚项。

因此，目前的数值：

[
J_{\mathrm{ISA}}=11004.536\ \mathrm{kg},
\qquad
J_{+10K}=10874.871\ \mathrm{kg}
]

只能表述为**当前简化温度修正模型下的计算结果**，不能作为完整非标准大气修正的最终结论。

---

# 一、做得合理的部分

## 1. 改用固定航程是正确的

第二问需要比较温度修正前后的总油耗。如果继续以

[
m(t_f)=62000\ \mathrm{kg}
]

作为终止条件，则油耗始终是：

[
m_0-m_f=10450\ \mathrm{kg}.
]

当前方案改为固定第一问参考航程：

[
X_{\mathrm{ref}}=200668.442\ \mathrm m,
]

再计算：

[
J=m_0-m(t_f),
]

这是合理的比较口径。

---

## 2. 标准大气基线方向正确

当前采用：

[
T=T_0-Lh,
]

[
p=p_0\left(\frac{T}{T_0}\right)^{g/(RL)},
]

[
\rho=\frac{p}{RT},
]

[
a=\sqrt{\gamma RT},
]

比第一问的“指数密度＋标准温度”混合模型更适合第二问。

---

## 3. 使用同一操作策略比较 ISA 和温度偏差是可以的

当前保持：

[
V=V_0,\qquad C_L=C_{L,\mathrm{ref}},
]

然后在不同大气模型下重新计算高度和油耗。

这是一个有效实验，但它的准确名称应是：

> 在相同等真空速、等参考升力系数操作规律下，比较不同大气条件中的固定航程油耗。

不能说两组实验保持了“同一飞行路径”，因为高度剖面已经重新计算并发生变化。结果文件也明确表示温度修正后重新计算了高度剖面。

# 二、最严重的问题：这不是“完整非标准大气修正”

方案中写的是：

> (T_{\mathrm{corr}}=T_{\mathrm{ISA}}+10K)，但压力剖面仍保留 ISA。

结果文档也明确承认：

> 压力剖面仍采用 ISA 压力。

这意味着实际采用的是：

[
\rho_{\mathrm{corr}}(h)
=======================

\frac{p_{\mathrm{ISA}}(h)}
{R[T_{\mathrm{ISA}}(h)+10]}.
]

但完整非标准大气应同时满足静力平衡：

[
\frac{dp_\Delta}{dh}
====================

-\rho_\Delta g,
\qquad
\rho_\Delta=\frac{p_\Delta}{RT_\Delta}.
]

因此应有：

[
p_\Delta(h)
===========

p_{\mathrm{ref}}
\exp\left[
-\int_{h_{\mathrm{ref}}}^{h}
\frac{g}{RT_\Delta(z)},dz
\right].
]

然后再计算：

[
\rho_\Delta(h)=\frac{p_\Delta(h)}{RT_\Delta(h)}.
]

## 恒定 (+10K) 时

若对流层温度为：

[
T_\Delta(h)=T_0+\Delta T-Lh,
]

并保持海平面压力 (p_0) 不变，则可以直接得到：

[
p_\Delta(h)
===========

p_0
\left(
\frac{T_0+\Delta T-Lh}
{T_0+\Delta T}
\right)^{g/(RL)}.
]

注意分母是：

[
T_0+\Delta T,
]

而不是原来的 (T_0)。

### 当前模型应如何定名

只有两种处理方式：

* **保留当前计算**：改名为“固定 ISA 压力的温度—密度简化修正”；
* **坚持完整修正**：重新计算 (p_\Delta,\rho_\Delta,a_\Delta)，然后刷新全部结果。

建议采用第二种。

# 三、声速实际上没有进入当前油耗方程

这是当前结论中的明显逻辑错误。

结果中写道：

> 温度通过密度和声速进入升力平衡、阻力、马赫数与燃油惩罚项。

但当前策略是：

[
V=240\ \mathrm{m/s},
]

燃油惩罚项是：

[
1+\beta(V-V_{\mathrm{opt}})^2.
]

因为 (V) 固定，所以：

[
1+\beta(240-235)^2
]

在所有温度场中完全相同。

声速只改变：

[
M=\frac{V}{a(h)},
]

但当前模型中：

* 没有马赫数相关波阻；
* 没有马赫数相关发动机效率；
* 没有等马赫速度约束；
* 燃油惩罚项依赖 (V)，不依赖 (M)。

所以在当前等真空速模型里：

> **声速只是诊断量，并不参与油耗计算。**

必须把结果中的这句话删除：

> 声速改变燃油惩罚项。

---

## 两种修复方案

### 方案 A：保留等真空速主模型

明确写：

> 声速连续变化用于计算马赫数和检查飞行状态，但在当前不含波阻、且空速固定的简化模型中，声速不直接进入油耗方程。

这在数学上诚实，但对题目中“声速如何进入模型”的回答较弱。

### 方案 B：加入等马赫对照

增加：

[
V(h)=M_0a(h).
]

此时声速会通过空速进入：

[
V_g=V+W(h),
]

[
D=\frac12\rho V^2SC_D,
]

[
1+\beta(V-V_{\mathrm{opt}})^2,
]

以及：

[
\frac{dV}{dt}
=============

M_0\frac{da}{dh}\frac{dh}{dt}.
]

**建议采用方案 B。**等真空速作为基线，等马赫作为体现声速连续场作用的主补充模型。

# 四、当前模型中密度也没有按结果文字所说直接改变阻力

当前保持：

[
C_L=C_{L,\mathrm{ref}},\qquad V=V_0,
]

并通过升力平衡反算高度：

[
\frac12\rho V^2SC_L=mg.
]

因此：

[
\frac12\rho V^2=\frac{mg}{SC_L}.
]

阻力为：

[
D
=

# \frac12\rho V^2SC_D

mg\frac{C_D}{C_L}.
]

由于 (C_L) 固定，所以：

[
C_D=C_{D0}+kC_L^2
]

也固定。

于是：

[
\boxed{D=mg\frac{C_D}{C_L}}
]

在给定质量下与温度、密度和高度没有直接关系。

因此当前 (+10K) 导致油耗变化的主要渠道不是“密度改变动压和阻力”，而是：

1. 温度修正改变 (\rho(h)) 的高度分布；
2. 为满足同一个 (C_L) 和 (V)，飞机需要采用不同的高度轨迹；
3. 高度—质量关系 (h(m)) 以及 (dh/dm) 改变；
4. 爬升功率项
   [
   \frac{mg}{V}\dot h
   ]
   改变；
5. 高度变化还会影响风速 (W(h))，进而改变完成固定地面航程所需的时间。

当前结果中的因果解释应按此重写。

# 五、还没有显式建立“沿路径积分模型”

当前方案实际求解的是：

[
\frac{dm}{dt},
\qquad
\frac{dx}{dt},
]

然后通过事件：

[
x(t_f)=X_{\mathrm{ref}}
]

终止。

这种数值方法可以使用，但论文中必须显式推导路径域形式：

[
dt=\frac{dx}{V_g},
]

[
J
=

# \int_0^{t_f}q_f(t),dt

\int_0^{X_{\mathrm{ref}}}
\frac{q_f(x)}{V_g(x)},dx.
]

即：

[
\boxed{
J=
\int_0^{X_{\mathrm{ref}}}
q_x(x),dx
}
]

其中：

[
q_x(x)
======

\frac{q_f(x)}
{V(x)+W(h(x))}
]

是单位航程油耗，单位为：

[
\mathrm{kg/m}.
]

对应的路径质量方程为：

[
\boxed{
\frac{dm}{dx}
=============

-\frac{q_f}
{V+W(h)}
}
]

否则现在更接近“固定航程终止的时间积分”，还没有充分体现题目要求的“沿飞行路径的数学模型”。

建议同时进行路径积分与时间积分交叉验证：

[
m_0-m_f
\stackrel{?}{=}
\int_0^{t_f}q_f,dt
\stackrel{?}{=}
\int_0^{X_f}q_x,dx.
]

# 六、单个 (+10K) 场景不能得出“何时可以忽略”

当前只计算：

[
\Delta T=0,\quad +10K.
]



但结果直接得出：

> 温度偏差远小于 (10K)，或者精度要求低于约百分之一时，可以忽略。

这个结论没有足够证据。

一个 (+10K) 点无法说明：

* 油耗变化是否与温差近似线性；
* 正温差和负温差是否对称；
* (+5K) 是否真的对应约 (0.5%)；
* 高度线性温差是否产生更强非线性；
* 忽略阈值具体在哪里。

至少需要计算：

[
\Delta T\in{-10,-5,-2,0,2,5,10}\ \mathrm K.
]

再定义：

[
\varepsilon_J(\Delta T)
=======================

\frac{|J(\Delta T)-J(0)|}{J(0)}.
]

根据数值寻找：

[
\varepsilon_J<1%
]

对应的温差范围。

还应加入随高度线性变化的场景：

[
\Delta T(h)
===========

\Delta T_0+\kappa(h-h_0).
]

# 七、“验证全部通过”表述过强

当前验证项目包括：

* 固定航程误差；
* 参数正性；
* 隐式 ODE 分母正性；
* 温度修正产生非零油耗变化。

前三项属于数值一致性检查。

但“温度修正产生非零变化”不是模型正确性的验证，因为错误实现同样可能产生非零变化。

建议补充：

| 验证项               | 含义                 |
| ----------------- | ------------------ |
| ISA 海平面值          | 检查标准大气实现           |
| 静力平衡残差            | 检查 (dp/dh=-\rho g) |
| (\Delta T\to0) 收敛 | 修正模型回到 ISA         |
| 时间/路径积分一致         | 检查积分转换             |
| 网格或步长敏感性          | 检查数值稳定性            |
| 正负温差测试            | 检查响应方向             |
| 小角度检查             | 检查准稳态假设            |
| 推力正性与范围           | 检查物理可行性            |
| 路径导数平滑性           | 检查高度反解稳定性          |

完整修正后，静力平衡相对残差可定义为：

[
r_{\mathrm{hyd}}
================

\frac{
\left|\dfrac{dp}{dh}+\rho g\right|
}{
\rho g
}.
]
