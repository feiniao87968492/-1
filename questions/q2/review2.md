## 总体评价

**这版已经修复了上一轮指出的主要错误，但仍不建议直接标记为 `done`。**

之前的几个 P0 问题确实已经解决：

* 非标准大气压力改为满足静力平衡；
* 显式建立了路径积分；
* 正确说明声速在当前等真空速模型中只是马赫数诊断量；
* 增加了正负多组温差；
* 补充了静力平衡、路径/时间积分一致性、步长敏感性等验证。 

但现在暴露出一个更隐蔽、影响全部数值结果的问题：

> **第一问的 (C_{L,\mathrm{ref}}) 与第二问的 ISA 密度模型不兼容，导致第二问没有从题设初始高度 (9500\ \mathrm m) 开始。**

---

# 一、当前最严重的问题：初始高度被静默改变

第二问目前的操作规律是：

[
V=240\ \mathrm{m/s},
\qquad
C_L=C_{L,\mathrm{ref}},
]

然后通过：

[
\frac12\rho(h)V^2SC_{L,\mathrm{ref}}=mg
]

反算高度。

第一问得到的参考升力系数是：

[
C_{L,\mathrm{ref}}=0.658914,
]

它是根据第一问的指数密度模型

[
\rho(h)=1.225e^{-h/7300}
]

在 (h_0=9500\ \mathrm m) 处计算出来的。

但是第二问已经改用了 ISA 密度。

将 (m_0=72450\ \mathrm{kg})、(V_0=240\ \mathrm{m/s}) 和原来的 (C_{L,\mathrm{ref}}) 代入升力平衡，需要的空气密度约为：

[
\rho_{\mathrm{req}}
===================

\frac{2m_0g}{V_0^2SC_{L,\mathrm{ref}}}
\approx0.33339\ \mathrm{kg/m^3}.
]

在 ISA 中，这个密度对应的高度不是 (9500\ \mathrm m)，而大约是：

[
h(0)\approx11679\ \mathrm m.
]

在 (+10\ \mathrm K) 修正大气中约为：

[
h(0)\approx11812\ \mathrm m.
]

所以当前程序实际上比较的是：

| 场景      |    实际反算初始高度 |
| ------- | ----------: |
| ISA     | 约 (11679) m |
| (+10) K | 约 (11812) m |
| 题设要求    |    (9500) m |

这意味着不同温度场不仅大气条件不同，**初始高度也不同**。因此现在的油耗差异并不是严格的同初始状态比较。

---

# 二、当前大气公式被用到了适用范围之外

目前采用：

[
T_\Delta(h)=T_0+\Delta T-Lh,
]

[
p_\Delta(h)
===========

p_0
\left(
\frac{T_0+\Delta T-Lh}
{T_0+\Delta T}
\right)^{g/(RL)}.
]

这个公式只适用于具有恒定温度递减率的对流层。

标准大气中，对流层上界约为 (11,000\ \mathrm m)。在其上，温度不再继续按 (T_0-Lh) 下降，而进入等温层。

根据当前结果：

[
m_f=72450-11004.536=61445.464\ \mathrm{kg}.
]

继续使用固定 (C_L) 和固定 (V) 反算，当前模型对应的高度范围大致为：

| 场景      |       初始高度 |       最终高度 |
| ------- | ---------: | ---------: |
| ISA     | 约 11.68 km | 约 12.92 km |
| (+10) K | 约 11.81 km | 约 13.11 km |

也就是说，**几乎整个第二问轨迹都在 11 km 以上，却仍然使用对流层递减温度公式。**

因此即使静力平衡公式本身正确，所采用的大气分层仍不正确。

---

## 正确的分层 ISA

### (0\le h\le11000\ \mathrm m)

[
T(h)=T_0-Lh,
]

[
p(h)
====

p_0
\left(\frac{T(h)}{T_0}\right)^{g/(RL)}.
]

### (11000<h\le20000\ \mathrm m)

令：

[
T_{11}=T_0-L\cdot11000,
]

则：

[
T(h)=T_{11},
]

[
p(h)
====

p_{11}
\exp
\left[
-\frac{g(h-11000)}{RT_{11}}
\right].
]

对于恒定温度偏差 (\Delta T)，上层温度应相应变为：

[
T_{11,\Delta}=T_{11}+\Delta T,
]

并以修正后的 (p_{11,\Delta}) 保证压力连续。

# 三、终止质量约束也被突破了

题目给定：

[
m_f=62000\ \mathrm{kg}.
]

当前 ISA 固定航程油耗为：

[
J=11004.536\ \mathrm{kg},
]

因此终点质量是：

[
m(X_{\mathrm{ref}})
===================

# 72450-11004.536

61445.464\ \mathrm{kg}.
]

这比题设终止质量低：

[
62000-61445.464
===============

554.536\ \mathrm{kg}.
]

当前结果表确实报告了 (11004.536\ \mathrm{kg}) 的基准油耗。

如果 (m_f=62000\ \mathrm{kg}) 是硬约束，那么当前参考航程在 ISA 模型下已经不可行。

这并不说明固定航程思路错误，而是参考航程的选取应调整。

## 更合理的参考航程

建议：

1. 在标准 ISA 条件下；
2. 从题设初始状态 (m_0,h_0,V_0) 出发；
3. 积分到 (m=62000\ \mathrm{kg})；
4. 将得到的地面航程记为 (X_{\mathrm{ISA}})；
5. 再让所有温差场景都飞相同的 (X_{\mathrm{ISA}})；
6. 比较不同温度场的终点质量和油耗。

这样：

* ISA 基准油耗恰好为 (10450\ \mathrm{kg})；
* 温度修正场景的油耗不再被固定；
* 固定航程比较仍然成立；
* 题设终止质量不会被基准场景突破。

# 四、推荐的模型修复方式

有两种可选方案。

## 方案 A：最小修改

对每一个大气场景重新计算初始升力系数：

[
C_{L,\mathrm{ref}}(\Delta T)
============================

\frac{
2m_0g
}{
\rho_\Delta(h_0)V_0^2S
}.
]

然后在该场景内保持：

[
C_L=C_{L,\mathrm{ref}}(\Delta T).
]

按当前参数：

| 场景      |  初始参考升力系数 |
| ------- | --------: |
| ISA     | 约 0.50052 |
| (+10) K | 约 0.49820 |
| (-10) K | 约 0.50384 |

这样可以保证所有场景均满足：

[
h(0)=9500\ \mathrm m.
]

但需要明确：

> 不同温度场使用不同的初始参考升力系数，比较的是“保持各自初始升力系数不变”的相同操作规则，而不是完全相同的 (C_L) 数值。

这是对现有代码改动最小的方案。

---

## 方案 B：更推荐，固定几何路径

直接使用第一问等速策略的参考路径：

[
h=h_{\mathrm{ref}}(x),
\qquad
V=V_{\mathrm{ref}}(x)=240\ \mathrm{m/s}.
]

在每种大气条件下，根据升力平衡动态计算：

[
C_L(x)
======

\frac{2m(x)g}
{\rho_\Delta(h_{\mathrm{ref}}(x))V^2S}.
]

然后：

[
C_D(x)=C_{D0}+kC_L(x)^2,
]

[
D(x)
====

\frac12\rho_\Delta(h_{\mathrm{ref}}(x))
V^2S C_D(x).
]

再沿同一条路径积分油耗。

### 优点

* 所有场景初始高度都是 (9500\ \mathrm m)；
* 所有场景经历相同的高度分布；
* 能真正隔离温度、密度变化对气动力和油耗的影响；
* 第一问等速路径只到约 (10637\ \mathrm m)，没有越过 11 km；
* 更符合题目“沿飞行路径经历连续大气参数场”的表述；
* 密度会真实改变 (C_L)、诱导阻力和总阻力，而不仅仅通过反算高度间接作用。

对于第二问，我更建议采用方案 B。
