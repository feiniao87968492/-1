# 一、Gate 2 的通过证据是充分的

## 1. 网格收敛成立

此前的速度重积分误差为：

[
0.030218
\rightarrow
0.007656
\rightarrow
0.001897
\rightarrow
0.0004806\ \mathrm{m/s}.
]

从 (N=121) 到 (N=241) 的误差比约为：

[
\frac{0.001897}{0.0004806}
\approx3.95,
]

仍然符合梯形法二阶收敛趋势。

质量重积分误差也由：

[
0.045053\ \mathrm{kg}
]

下降到约：

[
0.01066\ \mathrm{kg}.
]

这说明 (N=241) 不是通过偶然的数值波动，而是前面规则收敛趋势的自然延续。

## 2. ODE 容差不是误差主因

容差扫描得到：

[
\Delta V_f(10^{-8},10^{-10})
\approx3.22\times10^{-6}\ \mathrm{m/s},
]

[
\Delta V_f(10^{-10},10^{-12})
\approx4.55\times10^{-7}\ \mathrm{m/s}.
]

这些变化比 Gate 阈值 (10^{-3}\ \mathrm{m/s}) 小两到三个数量级，说明当前通过结果对 ODE 求解器容差稳定。

## 3. 不只检查了终点

本轮新增连续路径审计，检查了：

* 高度；
* 空速；
* 马赫数；
* 推力；
* 航迹角；
* 沿程状态重构误差。

连续约束违反为 0，因此不是只靠终端速度误差通过。

## 4. SLSQP 在 (N=241) 下仍然正常工作

大规模网格下 SLSQP 没有因规模问题失败，也没有放宽物理约束或 Gate 门槛。

这表明当前代码结构至少还能支撑下一阶段无风优化的首轮实现。

# 五、最终燃油优化必须保留的诊断

## 1. 连续重积分

继续保存：

* `reintegration_terminal_mass_error_kg`

* 连续路径约束违反

不能因为 Gate 2 已通过，就只检查离散 NLP。

## 2. 燃油恒等式

应验证：

报告：

=================

\left|

---------

\int_0^{X_f}

## 3. 网格目标收敛

至少报告：

[

\qquad
|J_{121}-J_{241}|.
]

通过标准应事先确定，例如绝对变化和相对变化都足够小。

## 4. 多初值

至少比较：

* 当前 (N=241) 可行解；
* Gate 1 轨迹；
* 平直或等速路径；
* 对高度、速度和控制施加小扰动的初值。

最终如果多个初值收敛到相同目标和相似轨迹，局部最优可信度会明显提高。

## 5. 求解器最优性

需要保存：

* solver success；
* first-order optimality；
* KKT residual；
* constraint violation；
* 迭代次数；
* 终止原因。

仅有“求解器成功退出”不足以证明最优性。

# 六、正式燃油优化前的两个核心模型风险

## 1. 自由终端时间可能产生过慢省油解

当前 (t_f) 自由，最终燃油优化可能通过降低速度或延长时间获得题面模型下的省油结果。

因此至少应同时报告：

### 无时间约束主结果

[
\min(m_0-m_f).
]

### 运营时间约束结果

例如：

[
t_f\le1.05,t_{\mathrm{base}},
]

和：

[
t_f\le1.10,t_{\mathrm{base}}.
]

无风基准时间为：

[
t_{\mathrm{base}}=790.755\ \mathrm s.
]

如果无约束最优时间显著延长，就不能只报告燃油节省而不讨论运营代价。

## 2. (T_{\min}=0) 可能产生零推力零油耗偏差

当前模型：

[
\dot m=-c_TT\Phi(V)
]

在 (T=0) 时给出零燃油流量。

最终最优结果必须报告：

[
T_{\min,\mathrm{traj}},
]

以及零推力或近零推力区间比例。

如出现长时间 (T\approx0)，至少需要增加一种对照：

* 正怠速推力 (T_{\mathrm{idle}}>0)；
* 基础燃油流量；
* 或明确声明这是题面简化模型造成的边界解。

* 高度、速度、马赫等路径约束乘子；
* 终端质量约束是否激活；
* 光滑非约束弧段上的 Hamiltonian/KKT 残差。

证据链中也明确说明 Hamiltonian 诊断尚未完成。
