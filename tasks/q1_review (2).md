# 二、必须修改：不要把 (\gamma) 称为直接物理控制量

目前推导中写的是：

> 实际物理控制量为推力 (T(t)) 与航迹角 (\gamma(t))。

这个表述不够严谨。

在当前模型中：

[
\dot h=V\sin\gamma,
]

因此：

[
\gamma=\arcsin\frac{\dot h}{V}.
]

一旦 (h(m))、(m(t)) 和 (V(m)) 已经确定，(\gamma) 是由轨迹反算出来的，并不是独立自由控制量。

更符合飞行动力学的说法是：

* 推力 (T) 是纵向操作量；
* 升力系数 (C_L) 或迎角 (\alpha) 是法向操作量；
* 航迹角 (\gamma) 是轨迹变量；
* 本问中通过固定 (C_L=C_{L,\mathrm{ref}}) 将法向操作策略固定下来。

建议统一改为：

> 状态或轨迹变量为 (x,h,V,m)，操作量通常可取推力 (T) 与升力系数 (C_L)（或迎角）。本问为闭合两类巡航爬升策略，令 (C_L=C_{L,\mathrm{ref}})，随后由高度变化率反算航迹角 (\gamma)。

这样不会和第三问的最优控制变量定义冲突。

# 四、需要修改：航程较大的原因表述不准确

当前结果写道：

> 等马赫策略……平均爬升率较低，但顺风累计航程更大。

这个说法容易让人理解为“主要是因为顺风更强，所以航程更大”。

但按当前模型拆分航程：

| 策略  |     空速累计距离 |    风场累计贡献 |      总地面航程 |
| --- | ---------: | --------: | ---------: |
| 等速  | 182.860 km | 17.808 km | 200.668 km |
| 等马赫 | 193.316 km | 18.016 km | 211.331 km |

两种策略的风场累计贡献只相差约：

[
18.016-17.808=0.208\ \mathrm{km}.
]

而总航程相差约：

[
10.663\ \mathrm{km}.
]

所以航程差异主要来自：

1. 等马赫策略耗油较慢；
2. 飞行时间更长；
3. 尽管空速逐渐下降，但累计空中距离仍更大；
4. 风场只提供较小的附加差异。

建议改成：

> 等马赫策略的空速随高度增加而下降，但其燃油消耗过程持续时间更长，因此累计空速航程仍高于等速策略；在相同顺风场的附加作用下，其最终地面航程更大。航程差异主要来自飞行时间，而非风速差异。

最好在结果表中增加：

* `air_distance_m`
* `wind_distance_contribution_m`
* `ground_distance_m`

这样风场作用会更清楚。

# 五、建议强化“等马赫爬升率较小”的数学解释

目前写的是：

> 原因是声速随高度降低改变升力平衡分母。

这个解释还不够完整。

## 等速策略

固定 (V) 和 (C_L) 时：

[
m\propto\rho(h),
]

因此：

[
\frac{d\ln m}{dh}
=================

-\frac1{H_\rho}.
]

于是：

[
\frac{dh}{dm}
=============

-\frac{H_\rho}{m}.
]

## 等马赫策略

由于：

[
V=Ma(h),
]

且：

[
a^2(h)=\gamma RT(h),
]

所以：

[
m\propto\rho(h)a^2(h).
]

若：

[
T(h)=T_0-\lambda h,
]

则：

[
\frac{d\ln m}{dh}
=================

-\frac1{H_\rho}
-\frac{\lambda}{T_0-\lambda h}.
]

第二项严格小于零，因此：

[
\left|
\frac{d\ln m}{dh}
\right|_{\text{等马赫}}

>

\left|
\frac{d\ln m}{dh}
\right|_{\text{等速}}.
]

从而在同样质量减少量下：

[
\left|
\frac{dh}{dm}
\right|*{\text{等马赫}}
<
\left|
\frac{dh}{dm}
\right|*{\text{等速}}.
]

物理意义是：

> 等马赫飞行时，高度增加不仅使密度下降，还使声速和真空速下降。速度下降本身会降低动压，因此不需要像等真空速策略那样爬升得那么高，就能在固定 (C_L) 条件下重新满足升力平衡。

此外，等马赫策略中 (V) 从 (240) m/s 下降到约 (236.75) m/s，逐渐接近 (V_{\mathrm{opt}}=235) m/s，使速度惩罚项：

[
1+\beta(V-V_{\mathrm{opt}})^2
]

减小，质量消耗速度变慢，进一步拉长飞行时间。因此本模型最终得到更低的时间平均爬升率。

这个解释应写进 `derivation.md` 或 `results.md`，而不只是说“改变分母”。
