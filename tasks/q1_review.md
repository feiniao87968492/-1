## 结论

**整体框架合理，但数学模型还不能直接作为第一问的最终执行方案。**

仓库产物、验证项目、脚本拆分和证据链设计都比较完整；主要问题集中在**模型闭合条件、评价指标和题目中“等马赫爬升率更大”的预设结论**。当前版本大约可以保留 70%，其余部分需要修改后再交给本地模型执行。

---

## 一、合理的部分

### 1. 正确识别了固定终止质量的问题

你已经意识到：

[
m_0-m_f=72450-62000=10450\ \mathrm{kg}
]

只要两种策略都以 (m_f=62000) kg 为终止条件，总燃油消耗就必然都是 (10450) kg。因此主要比较时间、航程、最终高度和爬升率，这是正确的。

不过应进一步处理为：

* **题目原始工况**：固定终止质量，比较时间、航程、高度；
* **补充公平工况**：固定航程，比较实际燃油消耗。

否则题目要求的“总燃油消耗量数值对比”只能得到两个完全相同的数字。

---

### 2. 拒绝直接使用 (T=D) 是正确的

巡航爬升过程中存在势能变化，等马赫策略还存在动能变化，因此不能简单写成：

[
T=D.
]

采用小航迹角下的能量模型：

[
T=D+m\dot V+\frac{mg}{V}\dot h
]

是合理的近似。你也正确指出了严格水平飞行和巡航爬升之间的矛盾。

但最好先从完整的小角度纵向方程开始：

[
\dot x=V\cos\gamma+W(h),
]

[
\dot h=V\sin\gamma,
]

[
m\dot V=T-D-mg\sin\gamma,
]

[
L=mg\cos\gamma.
]

再利用

[
\sin\gamma\approx \gamma\approx\frac{\dot h}{V},\qquad
\cos\gamma\approx1
]

推出：

[
T=D+m\dot V+\frac{mg}{V}\dot h.
]

这样逻辑更加完整，也能解释 (\gamma) 到底去了哪里。

---

### 3. 代码和产物规划基本合理

高度、质量、爬升率、高度—距离曲线，以及对应的生图 CSV 都已经规划，这是正确的证据链设计。

这部分可以直接保留。

---

## 二、最关键的问题：固定 (C_L) 是人为增加的闭合条件

当前方案写的是：

> 两种策略均保持初始升力系数 (C_L^*)。

这并不是由题目自动推出的，而是你额外加入的飞行策略假设。

因为仅有：

[
L=\frac12\rho V^2SC_L=mg
]

并不能同时确定 (h(t))、(V(t)) 和 (C_L(t))。指定等速或者等马赫后，还需要一个额外条件才能确定高度剖面。

保持 (C_L) 不变是可以采用的，但必须写成：

> 为闭合等速和等马赫巡航爬升模型，假设飞机通过连续调整迎角，使升力系数维持在参考值 (C_{L,\mathrm{ref}})。

还要解释为什么选这个值。

建议不要简单称为“初始升力系数”，而是计算：

[
C_{L,0}=
\frac{2m_0g}{\rho(h_0)V_0^2S}.
]

同时计算最大升阻比对应值：

[
C_{L,L/D_{\max}}=\sqrt{\frac{C_{D0}}{k}}.
]

题目参数下：

[
\sqrt{\frac{0.022}{0.045}}\approx0.699.
]

如果初始 (C_{L,0}) 与 0.699 接近，就可以说明：

> 初始工况接近最大升阻比状态，因此采用 (C_{L,\mathrm{ref}}=C_{L,0}) 作为巡航爬升闭合条件具有一定物理合理性。

这样才不是随意增加假设。

---

## 三、“等马赫策略爬升率更大”不能提前写死

这是当前方案最严重的问题。

在固定 (C_L) 条件下，由升力平衡：

[
m=\frac{\rho(h)V^2(h)SC_L}{2g}.
]

### 等真空速策略

因为 (V=V_0)，指数大气下：

[
\rho(h)=\rho_0e^{-h/H_\rho},
]

所以：

[
\frac{d\ln m}{dh}=-\frac1{H_\rho},
]

即：

[
\frac{dh}{dm}=-\frac{H_\rho}{m}.
]

因此可得到：

[
h(m)=h_0-H_\rho\ln\frac{m}{m_0}.
]

这部分是正确的。

---

### 等马赫数策略

等马赫条件为：

[
V(h)=M_0a(h).
]

于是：

[
m\propto \rho(h)a^2(h).
]

对数求导：

[
\frac1m\frac{dm}{dh}
====================

\frac1\rho\frac{d\rho}{dh}
+
2\frac1a\frac{da}{dh}.
]

在对流层内，温度随高度降低，因此：

[
\frac{da}{dh}<0.
]

所以：

[
\frac{d\ln m}{dh}
=================

-\frac1{H_\rho}
+
2\frac{d\ln a}{dh}
]

会比等速策略更负。这意味着在同样的质量下降量下，等马赫策略所需的高度增加量反而通常更小：

[
\left|\frac{dh}{dm}\right|*{\text{等马赫}}
<
\left|\frac{dh}{dm}\right|*{\text{等速}}.
]

也就是说，**仅从升力平衡看，你当前的固定 (C_L) 模型并不能推出等马赫策略爬升率更大，甚至很可能得到相反结果。**

从能量角度看，等马赫飞行过程中声速下降，真空速也下降：

[
\dot V<0.
]

在

[
T=D+m\dot V+\frac{mg}{V}\dot h
]

中，(m\dot V<0) 表示部分动能被释放，理论上可以支持爬升或降低推力需求。但这究竟会不会使时间爬升率 (\dot h) 更大，必须由完整隐式方程求解，不能预设。

因此方案中应该改成：

> 数值比较两种策略的爬升率。若结果与题面所述“等马赫策略爬升率更大”不一致，应检查闭合条件、大气模型和参数口径，并将不一致作为模型诊断结果，而不是人为调整结果以迎合题面。

---

## 四、大气模型目前不完全一致

当前设计是：

* 密度：指数大气；
* 声速：标准大气温度模型。

这属于混合大气模型。

不是绝对错误，但会造成：

[
\rho(h)
]

和

[
a(h)=\sqrt{\gamma RT(h)}
]

不是由同一套温度、压力关系推导出来的。

### 更好的处理方式

建议采用两层模型：

#### 解析推导模型

为获得等速策略的解析关系，使用：

[
\rho(h)=\rho_0e^{-h/H_\rho}.
]

用于推导：

[
h=h_0-H_\rho\ln(m/m_0).
]

#### 正式数值模型

统一采用分层国际标准大气：

[
T=T(h),\qquad p=p(h),
]

[
\rho(h)=\frac{p(h)}{RT(h)},
]

[
a(h)=\sqrt{\gamma RT(h)}.
]

这样等速和等马赫数值比较使用完全一致的大气模型。

## 五、阻力模型需要正面处理跨音速问题

当前仅使用：

[
C_D=C_{D0}+kC_L^2.
]

但初始速度是 (240) m/s，在约 9500 m 高度，马赫数大约处于 (0.79) 左右，已经接近典型民航客机的阻力发散区域。

因此不能完全不提跨音速阻力。

不过题目没有给出具体飞机的临界马赫数和波阻参数，因此不适合凭空建立复杂波阻主模型。

建议采用：

### 主模型

[
C_D=C_{D0}+kC_L^2.
]

理由：

* 题目只提供了 (C_{D0}) 和 (k)；
* 缺少可靠的波阻参数；
* 第一问重点是建立耦合框架。

### 扩展模型

增加平滑波阻项：

[
C_{D,\mathrm{wave}}
===================

k_w\left[\max(0,M-M_{\mathrm{crit}})\right]^n.
]

然后做参数敏感性分析，而不是把任意 (k_w) 当成真实飞机参数。

## 六、风场系数不能直接从 0.003 改成 (3\times10^{-5})

原题文本中给出：

[
W(h)=20+0.003(h-10000)^2.
]

你的方案则采用：

[
W(h)=20+3\times10^{-5}(h-10000)^2,
]

并称为“用户确认公式”。

这个修改在数值上很有必要，因为原公式在 (h=9500) m 时给出：

[
W=20+0.003\times500^2=770\ \mathrm{m/s},
]

显然不合理。

但不能只写成 OCR 错误，除非有原始清晰题面作为证据。

正确做法是同时保留两个场景：

| 场景        | 风场公式                           | 用途              |
| --------- | ------------------------------ | --------------- |
| Literal   | (20+0.003(h-10000)^2)          | 严格按题面，证明产生非物理风速 |
| Corrected | (20+3\times10^{-5}(h-10000)^2) | 合理性修正后的正式计算     |
| No-wind   | (W=0)                          | 验证和基线           |

然后在论文中说明：

> 原系数产生远超实际飞行速度的风场，疑似存在数量级错误，因此采用修正系数进行主要计算，并保留原公式作为异常场景对照。

## 七、(c_T) 也存在明显的数量级风险

题目给出：

[
c_T=0.00028\ \mathrm{kg/(N\cdot s)}.
]

假设巡航推力约为 (4\times10^4) N，则基础耗油率约为：

[
0.00028\times40000=11.2\ \mathrm{kg/s}.
]

这意味着每小时消耗约：

[
40320\ \mathrm{kg/h},
]

对于该质量级别飞机非常异常。

你已计划比较：

[
c_T=2.8\times10^{-5},
]

这个方向是正确的。

但正式方案应写成：

* 主结果严格使用题面 (2.8\times10^{-4})；
* 数量级修正场景使用 (2.8\times10^{-5})；
* 不静默替换题面参数；
* 对结果的工程合理性分别讨论。

## 八、残差标准不能统一写“绝对值小于 (10^{-6})”

当前写的是：

> 核心残差绝对值小于 (10^{-6})。

这不合理，因为：

* 升力残差单位为 N；
* 能量功率残差单位可能为 W；
* 质量终止误差单位为 kg；
* 数值量级完全不同。

应使用归一化残差。

### 升力平衡

[
r_L=
\frac{|L-mg|}{mg}.
]

### 能量方程

[
r_E=
\frac{
\left|TV-DV-mV\dot V-mg\dot h\right|
}{
\max(|TV|,|DV|,|mV\dot V|,|mg\dot h|,\varepsilon)
}.
]

建议标准：

[
r_L<10^{-6},\qquad r_E<10^{-5}.
]

终止质量单独使用：

[
|m(t_f)-m_f|<0.1\ \mathrm{kg}
]

或者根据求解容差设为 1 kg。

## 九、推荐修改后的主模型结构

### 基础动力学

[
\dot x=V\cos\gamma+W(h),
]

[
\dot h=V\sin\gamma,
]

[
m\dot V=T-D-mg\sin\gamma,
]

[
L=mg\cos\gamma,
]

[
\dot m
======

-c_TT\left[1+\beta(V-V_{\mathrm{opt}})^2\right].
]

### 气动力

[
L=\frac12\rho(h)V^2SC_L,
]

[
D=\frac12\rho(h)V^2S
\left(C_{D0}+kC_L^2+C_{D,\mathrm{wave}}\right).
]

### 小角度简化

[
L\approx mg,
]

[
T=D+m\dot V+\frac{mg}{V}\dot h.
]

### 策略闭合

等速：

[
V=V_0,\qquad C_L=C_{L,\mathrm{ref}}.
]

等马赫：

[
M=M_0,\qquad
V=M_0a(h),\qquad
C_L=C_{L,\mathrm{ref}}.
]

需要明确：

> 固定 (C_L) 是为了闭合巡航爬升策略而加入的操作假设，不是升力平衡自动给出的结果。
