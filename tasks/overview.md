## 建议采用的统一口径

题目同时把 (h(t),V(t))称为控制变量，又要求推导其时间演化，并给定终止质量 (m_f=62000\ \mathrm{kg})，确实存在数学定义和评价指标上的冲突。

建议不要逐字硬套，而是采用下面这套物理上闭合、数值上可实现的解释。

---

# 1. “水平飞行”解释为小角度准稳态巡航爬升

不能解释成严格的水平飞行。

严格水平飞行意味着

[
\gamma=0,\qquad \dot h=V\sin\gamma=0,
]

显然无法产生巡航爬升。

因此统一定义为：

> 飞机处于小航迹角、准稳态巡航爬升状态。其运动方向主要为水平方向，但允许高度缓慢变化。

使用近似

[
|\gamma|\ll 1,\qquad
\cos\gamma\approx 1,\qquad
\sin\gamma\approx \gamma,
]

于是

[
\dot x=V\cos\gamma+W(h)\approx V+W(h),
]

[
\dot h=V\sin\gamma,
]

[
L=mg\cos\gamma\approx mg.
]

这里的“水平运动方程”仅表示计算水平位移，不表示 (\dot h=0)。

---

# 2. 状态变量和控制变量重新定义

## 2.1 物理模型中的严格定义

建议把状态变量写成

[
\boldsymbol z(t)=
\begin{bmatrix}
x(t)&h(t)&V(t)&m(t)
\end{bmatrix}^{\mathrm T},
]

控制变量写成

[
\boldsymbol u(t)=
\begin{bmatrix}
T(t)&\gamma(t)
\end{bmatrix}^{\mathrm T}.
]

对应动力学为

[
\dot x=V\cos\gamma+W(h),
]

[
\dot h=V\sin\gamma,
]

[
m\dot V=T-D-mg\sin\gamma,
]

[
\dot m=-c_TT\left[1+\beta(V-V_{\mathrm{opt}})^2\right].
]

这是最标准、最不容易产生矛盾的点质量模型。

## 2.2 如何兼容题目“高度和空速为控制变量”的说法

在论文中可以说明：

> 题目中的“控制变量”是广义的轨迹决策变量，而不是严格意义上可以瞬时任意指定的物理输入。实际物理控制量为推力和航迹角，高度、空速则是在动力学约束下需要设计的状态轨迹。

在直接配点法或直接转录法中，每个节点上的

[
h_k,\quad V_k
]

确实都是优化决策变量，因此仍可称为“高度—速度联合控制轨迹”，但它们必须满足动力学约束，不能随意指定。

---

# 3. 推力和阻力不能在巡航爬升中始终令 (T=D)

纵向动力学为

[
m\dot V=T-D-mg\sin\gamma.
]

利用

[
\sin\gamma=\frac{\dot h}{V},
]

得到能量形式

[
\boxed{
T-D=m\dot V+\frac{mg}{V}\dot h
}
]

即

[
\boxed{
T=D+m\dot V+\frac{mg}{V}\dot h
}
]

其物理含义分别是：

[
TV
==

DV
+
mV\dot V
+
mg\dot h.
]

* (DV)：克服阻力所需功率；
* (mV\dot V)：改变动能所需功率；
* (mg\dot h)：增加势能所需功率。

因此：

* (T=D) 只适用于 (\dot h=0,\dot V=0) 的严格定高等速巡航；
* 等速巡航爬升时

[
T=D+\frac{mg}{V}\dot h>D;
]

* 变速巡航爬升时还必须考虑 (m\dot V)。

第一问的主模型应使用能量方程，不能一边令 (T=D)，一边又令飞机爬升。

---

# 4. 气动力模型采用小角度升力平衡

采用

[
L=\frac12\rho(h)V^2SC_L\approx mg,
]

所以

[
C_L=\frac{2mg}{\rho(h)V^2S}.
]

阻力极曲线取

[
C_D=C_{D0}+kC_L^2,
]

从而

[
D=\frac12\rho V^2S
\left(C_{D0}+kC_L^2\right).
]

消去 (C_L) 后可写成

[
\boxed{
D(h,V,m)
========

\frac12\rho(h)V^2SC_{D0}
+
\frac{2km^2g^2}{\rho(h)V^2S}
}
]

其中第一项为寄生阻力，第二项为诱导阻力。

第一问建议先采用该抛物线阻力模型。跨音速波阻应在第三、四问中作为附加项，例如

[
C_D=C_{D0}+kC_L^2+C_{Dw}(M),
]

同时设置

[
M\le M_{\max}.
]

否则第一问会因为缺少临界马赫数和波阻参数而引入新的任意参数。

---

# 5. 两种巡航爬升策略必须增加一个“闭合条件”

只有“等速”或“等马赫数”还不足以唯一确定高度。

例如等速只给出

[
V(t)=V_0,
]

但没有说明飞机为什么以及按照什么规律爬升。

由于题目没有给出发动机最大推力、固定油门或剩余推力曲线，建议采用：

> 巡航爬升过程中保持升力系数 (C_L=C_L^\ast) 不变。

取初始状态对应的升力系数

[
C_L^\ast
========

\frac{2m_0g}
{\rho(h_0)V_0^2S}.
]

使用题目给出的指数密度模型时，

[
\rho(h_0)
=========

1.225e^{-9500/7300}
\approx0.3334\ \mathrm{kg/m^3},
]

得到

[
C_L^\ast\approx0.659.
]

它与最大升阻比对应的

[
C_{L,L/D_{\max}}
================

# \sqrt{\frac{C_{D0}}k}

\sqrt{\frac{0.022}{0.045}}
\approx0.699
]

比较接近，因而具有一定合理性。

保持 (C_L) 基本不变，也符合巡航爬升过程中维持较高气动效率的思想。

---

# 6. 等真空速策略的高度演化规律

采用

[
V(t)=V_0,
\qquad C_L=C_L^\ast.
]

由升力平衡

[
mg=\frac12\rho(h)V_0^2SC_L^\ast.
]

与初始状态相除：

[
\frac{m(t)}{m_0}
================

\frac{\rho(h(t))}{\rho(h_0)}.
]

采用指数密度

[
\rho(h)=\rho_0e^{-h/H_\rho},
]

则

[
\frac{m(t)}{m_0}
================

e^{-[h(t)-h_0]/H_\rho}.
]

因此

[
\boxed{
h(t)=h_0-H_\rho
\ln\frac{m(t)}{m_0}
}
]

微分形式为

[
\boxed{
\dot h
======

-H_\rho\frac{\dot m}{m}
}
]

由于 (\dot m<0)，故 (\dot h>0)。

这说明巡航爬升的本质是：质量不断减小，为维持相同的 (C_L) 和空速，需要进入密度更低的高度。

若仅根据终止质量估算最终高度，

[
h_f
===

9500-7300
\ln\frac{62000}{72450}
\approx10635\ \mathrm{m}.
]

这个最终高度主要由升力平衡决定，与时间积分精度无关。

---

# 7. 等马赫数策略的高度演化规律

设

[
M(t)=M_0,
\qquad
M_0=\frac{V_0}{a(h_0)},
]

于是

[
V(t)=M_0a(h(t)).
]

升力平衡为

[
mg
==

\frac12\rho(h)M_0^2a^2(h)SC_L^\ast.
]

与初始状态相除：

[
\boxed{
\frac{m(t)}{m_0}
================

\frac{\rho(h(t))a^2(h(t))}
{\rho(h_0)a^2(h_0)}
}
]

由于理想气体中

[
a^2=\gamma RT,\qquad
p=\rho RT,
]

所以

[
\rho a^2=\gamma p.
]

因此还可以简化成

[
\boxed{
\frac{m(t)}{m_0}
================

\frac{p(h(t))}{p(h_0)}
}
]

即等马赫、等升力系数巡航爬升时，高度可以直接通过标准大气压力反算。

微分形式为

[
\frac{\dot m}{m}
================

\left(
\frac{\rho'(h)}{\rho(h)}
+
2\frac{a'(h)}{a(h)}
\right)\dot h,
]

所以

[
\boxed{
\dot h
======

\frac{\dot m/m}
{\mathrm d\ln\rho/\mathrm dh
+2,\mathrm d\ln a/\mathrm dh}
}
]

---

# 8. 不应预设“等马赫数爬升率一定更大”

题目要求解释“等马赫数策略爬升率更大”，但这不是一个普遍成立的物理结论。

在对流层中，温度随高度降低，因此

[
a(h)=\sqrt{\gamma RT(h)}
]

通常也随高度降低，即

[
\frac{\mathrm da}{\mathrm dh}<0.
]

于是等马赫策略的分母为

[
\frac{\mathrm d\ln\rho}{\mathrm dh}
+
2\frac{\mathrm d\ln a}{\mathrm dh},
]

其绝对值通常比等真空速策略的

[
\frac{\mathrm d\ln\rho}{\mathrm dh}
]

更大。在相同相对质量消耗率下，往往得到

[
\dot h_{\text{等马赫}}
<
\dot h_{\text{等真空速}}.
]

当然，两种策略的阻力、推力和耗油率不同，(\dot m) 也不同，因此最终结果必须数值计算后确定；但不能把“等马赫爬升率更大”当成预先必须满足的结论。

论文中的稳妥写法应是：

> 比较两种策略的爬升率，并检验题目所述等马赫策略具有更大爬升率的结论是否在给定参数和所选模型下成立。若不成立，则从声速随高度下降以及升力平衡关系解释其原因。

不要为了迎合题面结论而人为修改计算结果。

# 9. 质量微分方程实际上是隐式的

燃油消耗率取

[
\dot m
======

-c_TT\Phi(V),
]

其中

[
\Phi(V)=1+\beta(V-V_{\mathrm{opt}})^2.
]

而推力为

[
T
=

D+m\dot V+\frac{mg}{V}\dot h.
]

在两种策略下，(h) 和 (V) 都可表示成质量的函数：

[
h=h(m),\qquad V=V(m).
]

因此

[
\dot h=\frac{\mathrm dh}{\mathrm dm}\dot m,
\qquad
\dot V=\frac{\mathrm dV}{\mathrm dm}\dot m.
]

代入推力方程：

[
T
=

D+
\left(
m\frac{\mathrm dV}{\mathrm dm}
+
\frac{mg}{V}\frac{\mathrm dh}{\mathrm dm}
\right)\dot m.
]

定义

[
A(m)
====

m\frac{\mathrm dV}{\mathrm dm}
+
\frac{mg}{V}\frac{\mathrm dh}{\mathrm dm},
]

则

[
\dot m=-c_T\Phi(V)[D+A\dot m].
]

可以直接解出

[
\boxed{
\dot m
======

-\frac{c_T\Phi(V)D}
{1+c_T\Phi(V)A}
}
]

这样就能将每种策略最终化为一个关于 (m(t)) 的一维常微分方程，再由 (m(t)) 重构 (h(t))、(V(t))、(T(t)) 和 (x(t))。

这比在代码中同时积分多个互相循环依赖的变量更稳定。

# 10. 终止质量固定时，总燃油消耗不能用于比较

题目给出

[
m_0=72450\ \mathrm{kg},
\qquad
m_f=62000\ \mathrm{kg}.
]

因此无论采用什么策略，

[
J=m_0-m_f=10450\ \mathrm{kg}.
]

所以第一问要求比较“总燃油消耗量”时，只能得到

[
J_{\text{等速}}
=============

# J_{\text{等马赫}}

10450\ \mathrm{kg}.
]

这不是数值模型的结果，而是终止条件直接规定的。

## 第一问应比较的指标

在终止质量相同的条件下比较：

[
h_f,\qquad
t_f,\qquad
x_f,\qquad
\bar{\dot h},\qquad
\max \dot h.
]

其中最有区分度的是：

* 消耗相同燃油后飞行的地面航程；
* 达到终止质量所需时间；
* 最终高度和爬升率分布。

题目背景本身也提出了两种合理形式：固定航程最小化油耗，或固定燃油最大化航程。

# 11. 各问建议采用不同的终止口径

## 问题一：固定燃油

终止条件：

[
m(t_f)=62000\ \mathrm{kg}.
]

比较：

[
t_f,\quad x_f,\quad h_f.
]

总耗油统一报告为

[
10450\ \mathrm{kg},
]

同时说明该指标没有策略区分度。

## 问题二：固定航程

先用问题一中的等速基准策略得到

[
X_{\mathrm{ref}}
================

x_{\text{等速}}(t_f).
]

问题二改为以

[
x(t_f)=X_{\mathrm{ref}}
]

为终止条件，计算

[
J=m_0-m(t_f).
]

这样温度偏差修正后，总燃油消耗才会发生变化。

否则仍然固定 (m_f=62000)，无论怎样修正温度模型，总油耗永远都是 (10450\ \mathrm{kg})，只能改变时间和航程。

## 问题三：固定航程、最小油耗

目标函数为

[
\min J=m_0-m(t_f)
]

或等价地

[
\max m(t_f),
]

约束

[
x(t_f)=X_{\mathrm{ref}}.
]

## 问题四：进行两套等价比较

### 固定航程比较燃油节省率

[
\eta_f
======

\frac{J_{\mathrm{base}}-J_{\mathrm{opt}}}
{J_{\mathrm{base}}}\times100%.
]

### 固定燃油比较航程增益

[
\eta_x
======

\frac{X_{\mathrm{opt}}-X_{\mathrm{base}}}
{X_{\mathrm{base}}}\times100%.
]

这样“燃油节省比例”和“航程变化”都有明确含义，不会混在同一个终止条件中。

# 12. 题目参数中还有两个需要提前处理的问题

## 12.1 风场公式的量级明显异常

题目给出

[
W(h)=20+0.003(h-10000)^2\ \mathrm{m/s}.
]

若 (h) 以米计，在初始高度 (9500\ \mathrm m)：

[
W(9500)
=======

# 20+0.003\times500^2

770\ \mathrm{m/s}.
]

这已经远超正常高空风速，甚至超过飞机空速，不能直接用于计算。

建议改成带尺度的形式：

[
\boxed{
W(h)
====

20+
3\left(\frac{h-10000}{1000}\right)^2
\ \mathrm{m/s}
}
]

等价于

[
W(h)=20+3\times10^{-6}(h-10000)^2.
]

此时

[
W(9500)=20.75\ \mathrm{m/s},
]

数量级合理。

还需规定：

[
W>0
]

表示顺航向风，因此

[
\dot x=V\cos\gamma+W(h).
]

逆风情形直接令 (W<0)，不要在不同程序中反复切换加减号。

## 12.2 (c_T) 的数量级可能偏大

给定

[
c_T=2.8\times10^{-4}\ \mathrm{kg/(N\cdot s)}.
]

若巡航推力为 (3\times10^4\ \mathrm N)，即使不考虑速度惩罚，也有

[
\dot m_f\approx8.4\ \mathrm{kg/s},
]

会导致约 (10^4\ \mathrm{kg}) 燃油在很短时间内耗尽。

建议：

* 主结果仍使用题目原始 (c_T)，保证遵从题设；
* 增加 (c_T=2.8\times10^{-5}) 的工程合理性对照；
* 不要在没有说明的情况下直接替换；
* 所有结果表必须注明采用的是“题设参数”还是“修正参数”。

# 最终建议写进项目假设文件的口径

| 项目         | 固定口径                               |
| ---------- | ---------------------------------- |
| 飞行状态       | 小航迹角准稳态巡航爬升，不是严格水平飞行               |
| 物理状态变量     | (x,h,V,m)                          |
| 物理控制变量     | (T,\gamma)                         |
| 高度、速度的地位   | 优化中的轨迹决策变量，但受动力学约束                 |
| 升力关系       | (L\approx mg)                      |
| 推力关系       | (T=D+m\dot V+mg\dot h/V)           |
| 第一问闭合条件    | 保持 (C_L=C_L^\ast=C_L(h_0,V_0,m_0)) |
| 等速定义       | 恒定真空速 (V=V_0)                      |
| 等马赫定义      | 恒定 (M=M_0)，(V=M_0a(h))             |
| 第一问终止条件    | (m_f=62000\ \mathrm{kg})           |
| 第一问主要比较量   | 航程、时间、最终高度、爬升率                     |
| 第一问总油耗     | 两者均为 (10450\ \mathrm{kg})，无区分度     |

按照这套口径，第一问会形成一个闭合的微分—代数系统，第二至第四问的评价指标也不会因为固定终止质量而失去意义。
